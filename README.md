# HexWolf Portfolio

HexWolfのポートフォリオサイトです。プロフィールやこれまでの制作物を紹介しています。

- 公開URL: https://portfolio.hexwolf.org
- 姉妹サイト(ブログ): https://blog.hexwolf.org

## 技術スタック

- **SSG**: [Astro](https://astro.build)
- **UIライブラリ**: [React](https://react.dev)
- **スタイル**: [Tailwind CSS v4](https://tailwindcss.com)
- **プロジェクト管理**: Astro Content Collections

## 開発

```sh
npm install
npm run dev
```

| Command           | Action                                       |
| :----------------- | :-------------------------------------------- |
| `npm install`      | 依存パッケージのインストール                 |
| `npm run dev`       | ローカル開発サーバー起動 (`localhost:4321`) |
| `npm run build`     | 本番用ビルドを `./dist/` に出力              |
| `npm run preview`   | ビルド結果をローカルでプレビュー             |
| `npm run astro ...` | `astro add` などのAstro CLIコマンドを実行    |
