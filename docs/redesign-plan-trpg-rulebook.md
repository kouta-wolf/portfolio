# リデザイン計画: TRPGルールブック風ダークテーマ

ブレインストーミングの整理メモ。実装前の方針合意用。

## テーマ

TRPGのルールブックをモチーフにしたダークテーマ + レイアウト抜本改修。

## カラー

3色構成に削減(`primary`/`secondary`は廃止)。

```css
--color-bg: oklch(0.12 0.08 264);     /* 蒼黒 #000023 (背景) */
--color-text: oklch(0.97 0 0);        /* 銀白 #f6f6f6 (文字) */
--color-accent: oklch(0.78 0.09 220); /* 月光色 #8fc6dd (リンク/ボタン/見出し/ボーダー、旧accentを昇格) */
```

### 二層構成(背景は暗く、本の中身は紙質)

「ルールブック=本」のモチーフを保つため、画面全体を無機質なダークUIにせず、以下の二層で分ける。

- **外側(画面全体の背景)**: 上記`--color-bg`蒼黒のまま。暗闇に本が浮かぶ演出。
- **内側(本文カラム・カード・モーダルなど「本の中身」に相当する領域)**: 和紙/羊皮紙テクスチャ + オフホワイト〜生成り系の背景色、それに合わせた濃色の本文文字色を別途用意する。

```css
--color-paper-bg: oklch(0.94 0.015 75); /* 生成り/羊皮紙 (本文カラム・カードの背景。paper_00084.jpgの色味に合わせた確定値) */
--color-paper-text: oklch(0.24 0.02 50); /* 本文用の濃色文字 (紙背景上で使用。確定値) */
```

紙テクスチャ画像は`src/assets/paper_00084.jpg`を採用(グレーがかったオフホワイトの紙粒子感のみで、シミ等の汚れなし。真っ白すぎない点も条件に合致)。

敷き方: `background-image: url(paper_00084.jpg)` + `background-size: cover`を本文カラム・カード・モーダルの背景に適用し、上から`--color-paper-bg`を`background-blend-mode: multiply`で薄く重ねて色を均一化する(サイズの異なる要素に同じ画像を敷いた際の継ぎ目・色ムラを抑えるため)。見出しやサイドマージンの挿絵アイコンなど「紙の上」に乗らない要素は背景側(蒼黒)の扱いのままとする。

トレードオフ: 3色ミニマル構成から本文カラーのみ例外的に増える。「本の中身だけ紙」と役割を限定することで破綻を避ける。

## レイアウト

- 1画面1セクション(Hero, About, Works, Social, Contact等)。`min-h-screen` + `scroll-snap-type: y`
- 右下に「ページをめくる」ビジュアル(折り目風)を固定配置し、次セクションへ誘導
- Hero内にリンクを設置し、各セクションへ直接ジャンプ可能にする
- 本文は非対称2カラム: 本文カラム(狭め・行間広め、密度を上げない) + サイドマージンカラム(挿絵アイコン・一言フレーバーテキストのみ、余白多め)。Tailwind標準の`md`(768px)ブレークポイントで1カラムに崩す

### Portfolio(Works)セクションのみ例外

- 内部スクロールは設けない。1画面(1スナップ単位)に作品2つ = 見開きページとして扱い、他セクション(Hero/About等)と同じ`scroll-snap-type: y mandatory`にそのまま乗せる(外側との衝突を避けるための特別な調整は不要)
- 作品数が2の倍数でない/多い場合は複数の見開きページに分割。現状は作品数がそれほど多くないためこの方式で収まる見込み。将来的に増えた場合はカテゴリ大別のうえ再検討する
- 掲載作品は全件ではなく厳選する(細かい作品まで含めて全部載せることはしない)
- プロジェクトはカード形式(画像+テキスト)
- クリックで**モーダル**表示(別ページ遷移・別タブは不採用。理由: 1画面完結の没入感を優先)
- モーダルは上記「紙レイヤー」を採用し、和紙/羊皮紙が開くような質感・トランジションにする(背景の暗さと対比させる)
- モーダル自体はオーソドックスな中央配置ダイアログ + 背景オーバーレイ暗転。閉じる導線は「上部の戻る/バツボタン」または「周囲の暗転部分をクリック」の2通り

## セクション名(章立て表記)

素直な日本語 + 章番号でルールブック感を出す。TRPG専門用語はあえて使わない(初見での分かりにくさを避ける)。

- 第一章 プロフィール (About)
- 第二章 作品集 (Works/Portfolio)
- 第三章 連絡先 / つながり (Social)
- 終章 お問い合わせ (Contact)

ヘッダー常設ナビは短縮形、目次(下記)はフル表記、で統一しておく。

## 目次(TOC)

- 左端に常時薄く見える「しおり(リボン)」タブを配置。色は`--color-accent`(月光色)、アイコンは栞のタッセル風
- ホバーでページをめくるように目次がスライドインし、章タイトル(フル表記)一覧を表示。トランジションは200〜300ms ease-out
- モバイル(ホバー不可)はタップでスライドイン/アウトをトグルする方式にする
- 初見でナビ導線に迷わないよう、**ヘッダーにも最低限の常設ナビ**(プロフィール/作品集/連絡先/お問い合わせ)を併設
